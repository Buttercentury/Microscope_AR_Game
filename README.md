# Microscope · ปริศนากล้องจุลทรรศน์

เกมการศึกษาแบบ AR ฟันคำตอบผ่านเว็บแคม สำหรับนักเรียนชั้นมัธยมศึกษาปีที่ 4 (ชีววิทยา · เรื่องกล้องจุลทรรศน์)

## เล่นเกม

👉 **https://USERNAME.github.io/microscope-ar-game/**

> แก้ `USERNAME` เป็นชื่อผู้ใช้ GitHub ของคุณ หลังเปิด GitHub Pages แล้ว

## วิธีเล่น

1. เปิดลิงก์ด้วย **Google Chrome** หรือ **Microsoft Edge** บนคอมพิวเตอร์
2. กด **"อนุญาต" (Allow)** เมื่อเบราว์เซอร์ขอใช้กล้อง
3. ยืนห่างจอประมาณ **1 เมตร** ให้เห็นมือและแขนในกล้อง
4. อ่านคำถามด้านบน **15 วินาที**
5. คำตอบจะลอยขึ้นมาทีละอัน — **ใช้มือฟันคำตอบที่ถูก**

| การกระทำ | คะแนน |
|---|---|
| ฟันคำตอบถูก | **+10** |
| ฟันคำตอบผิด | **−5** |

- 1 รอบ = 10 ข้อ (สุ่มจากคลัง 40 ข้อ)
- แต่ละข้อมี 5 ตัวเลือก มีคำตอบถูกเพียงข้อเดียว
- ตำแหน่งตัวเลือกสลับใหม่ทุกครั้งที่เล่น

## ข้อกำหนด

- Chrome หรือ Edge (เดสก์ท็อป/โน้ตบุ๊ก)
- กล้องเว็บแคม
- ต้องต่ออินเทอร์เน็ต (โหลด MediaPipe จาก CDN)
- แสงสว่างเพียงพอ ฉากหลังไม่รก จะตรวจจับมือได้แม่นขึ้น

## ความเป็นส่วนตัว

ภาพจากกล้อง **ประมวลผลในเครื่องผู้เล่นทั้งหมด** ไม่มีการอัปโหลดหรือบันทึกวิดีโอไปที่ใด

## เอกสารประกอบ

[`Microscope_Quiz_Questions.pdf`](Microscope_Quiz_Questions.pdf) — คำถามทั้ง 40 ข้อ พร้อมเฉลยและคำอธิบาย สำหรับครูใช้ทบทวนก่อน/หลังเล่น

## ปรับแต่งเกม

แก้ค่าใน `CFG` ที่อยู่ในไฟล์ `index.html`

| ค่า | ค่าเริ่มต้น | ความหมาย |
|---|---|---|
| `roundQuestions` | 10 | จำนวนข้อต่อรอบ |
| `readTime` | 15 | วินาทีสำหรับอ่านคำถาม |
| `hangTime` | 8.0 | วินาทีที่คำตอบลอยอยู่บนจอ |
| `cardStagger` | 3000 | ระยะห่างการลอยแต่ละใบ (มิลลิวินาที) |
| `timePerQuestion` | 22 | วินาทีสำหรับตอบ |
| `minSliceSpeed` | 520 | ความเร็วมือขั้นต่ำที่นับเป็นการฟัน (px/s) |

> เปลี่ยน `hangTime` ได้เลย ระบบจะคำนวณแรงโน้มถ่วงและการเบนของการ์ดให้อัตโนมัติทุกขนาดจอ
